Buscamos

Através de dados históricos fornecidos pela CET e pela colaboração dos usuários, informar de forma simples e fácil se o local indicado pela sua geolocalização apresenta pontos de alagamento

API

Ocorrências

Buscar ocorrências baseado na geolocalização

						GET: /ocorrencias/{latitude}/{longitude}
							## -> /ocorrencias/-23.5555392/-46.5773751
							{
								   quantidadeCET: 1
								   quantidadeUsuario: 0
								   latitude: -23.6452
								   longitude: -46.6645
							}

Ocorrências

Reportar ocorrências baseado na geolocalização

						POST: /ocorrencias
							## -> /ocorrencias
							{
								   latitude: -23.5555392
								   longitude: -46.5773751
								   nivel: 'LEVE|MEDIO|ALTO'
							}

Previsão de chuva

Buscar previsão de tempo [São Paulo]

						GET: /saopedro/previsao
							## -> /saopedro/previsao
							{
								   code: "11" ## Vai chover
								   date: "22 Mar 2014"
								   day: "Sat"
								   high: "24"
								   low: 	"18"
